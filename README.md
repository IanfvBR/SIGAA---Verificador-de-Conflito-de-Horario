<h1>SIGAA - Verificador de Conflito de Horário</h1>
<h2>Descrição</h2>
<p> Uma extensão (add-on) que automaticamente sinaliza disciplinas que geram conflito de horário com &#x274C;,
e as que não geram conflito de horário com &#x2705; durante o processo de matrícula ou rematrícula em disciplinas.
</p>
<h2>Informações Técnicas</h2>
<p>&bullet; O código só é executado quando o usuário clica no ícone da extensão.</p>
<p>&bullet; O código apenas interage com elementos da página, sem realizar qualquer comunicação com servidores ou sites.</p>
